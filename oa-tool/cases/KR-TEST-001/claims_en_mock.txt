CLAIMS (English Translation)

1. A method for measuring a position, comprising:
   collecting a plurality of wireless signals;
   calculating a first positioning reference value based on the collected wireless signals; and
   determining a position by fusing the first positioning reference value with inertial sensor data.

2. The method of claim 1, wherein the wireless signals include at least one of Wi-Fi signals, Bluetooth signals, and cellular signals.

3. The method of claim 1, wherein the first positioning reference value is a weighted average of Received Signal Strength Indication (RSSI), Round-Trip Time (RTT), and Angle of Arrival (AoA).

4. The method of claim 1, wherein the inertial sensor data includes data from an accelerometer and a gyroscope.

5. The method of claim 1, further comprising:
   re-calculating the first positioning reference value upon occurrence of a handover event between adjacent base stations.

6. A terminal device comprising:
   a wireless signal collector configured to collect a plurality of wireless signals;
   a first positioning reference value calculator configured to calculate a first positioning reference value; and
   a position determiner configured to determine a position based on the first positioning reference value.

7. The method of claim 1, wherein the first positioning reference value is dynamically adjusted according to a user movement pattern.

8. The terminal device of claim 6, further comprising an inertial sensor.
